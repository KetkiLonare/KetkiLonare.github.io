🔑 Key Rules to Remember
⚙️

Backend .env Change

Only needs container restart. No rebuild.

docker compose down && docker compose up -d

🖼️

Frontend .env Change

VITE_ variables bake at build time. Requires full image rebuild.

docker compose build --no-cache

🌐

Nginx Config Change

Always test before reload. Zero downtime reload available.

nginx -t && systemctl reload nginx
